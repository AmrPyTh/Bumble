Versatile Mobile Manipulator Robot

Innovative Solution for Smart Logistics

Project Overview

This project presents a multifunctional mobile manipulator robot capable of remote control, autonomous movement, and real-time interaction via a dedicated mobile application. The solution is optimized for logistics and warehouse environments.

Application Screens

Key Features

  • Manual and Autonomous Navigation
  • Robotic Arm Integration
  • Real-Time Monitoring via Mobile Interface
  • ESP32 Server Connectivity
  • Emergency Notifications via Telegram
  • Object Detection & Voice Interaction

Technologies Used

  • Flutter for Cross-Platform Mobile App
  • ESP32 Microcontroller with Arduino
  • Telegram Bot API
  • Computer Vision Modules
  • Natural Language Processing (Voice Control)

Applications

  • Smart Warehousing
  • Emergency & Rescue Missions
  • Factory Automation
  • Educational & Research Prototypes
